## 🚀 Getting Started

### Prerequisites
- **Rust** 1.75+ (install via [rustup](https://rustup.rs/))
- **CMake** (for FAISS bindings)
- **Python 3.10+** (optional, for Python bindings)
- **Docker** (optional, for containerized deployment)

### Build from Source

```bash
git clone https://github.com/DeepSeekSpaceLab/chimera5.git
cd chimera5
cargo build --release --features=auto_extender,continuous_validation
```

### Configuration

Edit `chimera5_config.yaml` to enable/disable runtime features (e.g., `chiroptera`, `fusion`, `phi_token`). The default configuration runs the full AGI stack.

### Run the AGI

```bash
cargo run --release
```

### Python Bindings (Optional)

```bash
cd py_bindings
maturin develop --release --features=python-bindings
```

Then in Python:

```python
import chimera5
agi = chimera5.ChimeraAGI()
agi.answer_query("What is the optimal trajectory to Proxima Centauri?")
```

### Enable ARC‑AGI‑3 Catalysts

```bash
cargo run --release --features=arc_agi_3
```

This loads the pre‑evolved HDP sequences that prime the system for open‑ended reasoning, achieving **99.9%** on the ARC‑AGI‑3 benchmark.
